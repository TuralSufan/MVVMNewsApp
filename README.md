# MVVMNewsApp

#### MVVMNewsApp is a simple news app in MVVM architecture working with public [NewsApi](https://newsapi.org).
---

## **Modules**

    * MVVM architecture
    * Home Feed containing Recent News
    * Saved News articles
    * Search news articles 
    * Pagination

## **Libraries**

     * Room Persistence Library
     * AndroidX
     * LiveData
     * Navigation component
     * Retrofit
     * Glide
