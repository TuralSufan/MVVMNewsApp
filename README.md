# MVVMNewsApp

#### MVVMNewsApp is a simple news app in MVVM architecture working with public [NewsApi](https://newsapi.org).
---

## **Best practices**

* Navigation Components
* Kotlin Coroutines
* Retrofit
* Room Persistence Library
